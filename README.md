# Ataraxia

Orchestator for massively parallelized and immutable backtests.

[![License: Apache 2.0](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](LICENSE)

## What it is

Event-driven orchestrator for massively parallelized backtests of trading strategies. It runs your strategy while capturing each computation step for inspection.

### Development quickstart

```sh
git clone https://github.com/msitarz/ataraxia
cd ataraxia
make setup
```
